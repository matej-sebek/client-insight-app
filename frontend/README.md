# Frontend

Frontend část školního projektu **Evidence klientských poznatků** pro Unicorn University.

Použité technologie:

- React
- Vite
- TypeScript
- Tailwind CSS 4
- shadcn/ui styl komponent

## Spuštění

```bash
npm install
npm run dev
```

Frontend běží na:

```text
http://localhost:5173
```

Backend musí běžet na:

```text
http://localhost:3000
```

API base URL je nastavená přes:

```text
VITE_API_BASE_URL=http://localhost:3000
```
